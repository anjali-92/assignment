## Repo
To clone repository. 
```bash
$ git clone https://github.com/anjali-92/assignment.git
```

Go the directory. 
```bash
$ cd assignments
```

## Run tests
```bash
$ pytest
```

## To test code interactively
a) Sort tuples with priority. 
```bash
$ python src/tuple_sort.py
```

b) Binary Search Tree. 
```bash
$ python src/binary_search_tree.py
```
